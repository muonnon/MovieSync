## 프로젝트 소개

MovieSync는 영화진흥위원회 API를 활용한 **실시간 박스오피스 정보 제공** 및 **사용자 간 영화 토론 플랫폼**입니다.

사용자들은 실시간 박스오피스 Top 10 영화를 조회하고, 각 영화별 채팅방에서 다른 사용자들과 소통하며, 감상평과 별점을 공유할 수 있습니다.

### 시스템 구성도

```
┌─────────────────┐     TCP Socket     ┌─────────────────┐
│                 │    (Port 55555)    │                 │
│  Client (GUI)   │ ←────────────────→ │     Server      │
│   - Swing UI    │                    │  - Multi-thread │
│   - 로그인      │                    │  - DB Manager   │
│   - 영화 목록   │                    │  - API Manager  │
│   - 채팅        │                    │                 │
│   - 감상평      │                    └────────┬────────┘
│   - 북마크      │                             │
└─────────────────┘                             │
                                                │
                    ┌───────────────────────────┼───────────────────────────┐
                    │                           │                           │
                    ▼                           ▼                           ▼
          ┌─────────────────┐         ┌─────────────────┐         ┌─────────────────┐
          │   SQLite DB     │         │  영화진흥위원회  │         │    ChatRoom     │
          │  - Users        │         │   Open API      │         │  - 영화별 채팅  │
          │  - Movies       │         │  (박스오피스)   │         │  - 실시간 통신  │
          │  - Reviews      │         └─────────────────┘         └─────────────────┘
          │  - Bookmarks    │
          └─────────────────┘
```

## 주요 기능

### 🔐 사용자 인증
- 닉네임 기반 로그인 (회원가입 불필요)
- 닉네임 중복 검사

### 🎬 영화 정보
- 실시간 박스오피스 Top 10 조회
- 영화 상세 정보 (순위, 개봉일, 누적관객, 누적매출)
- 영화 제목 검색 기능

### 💬 실시간 채팅
- 영화별 전용 채팅방
- 입장/퇴장 알림
- 실시간 메시지 브로드캐스트

### ⭐ 감상평 시스템
- 별점 (1~5점) 및 텍스트 감상평 작성
- 평균 평점 계산
- 본인 작성 감상평 삭제

### 🔖 북마크
- 관심 영화 북마크 추가/삭제
- 북마크 목록 조회

## 시스템 요구사항

### 필수 환경
- **Java**: JDK 8 이상
- **데이터베이스**: SQLite (내장, 별도 설치 불필요)
- **네트워크**: TCP 포트 55555 사용

### 외부 라이브러리
- `sqlite-jdbc` - SQLite JDBC 드라이버

### API
- [영화진흥위원회 오픈 API](https://www.kobis.or.kr/kobisopenapi/homepg/main/main.do) - 일일 박스오피스 조회

## 설치 및 실행

### 1. 라이브러리 설정
`lib/` 폴더에 SQLite JDBC 드라이버(`sqlite-jdbc-x.xx.x.jar`)를 배치합니다.

### 2. 서버 실행
```bash
# 컴파일
javac -cp lib/sqlite-jdbc-*.jar -d bin src/*.java

# 서버 실행
java -cp "bin;lib/sqlite-jdbc-*.jar" Server
```

### 3. 클라이언트 실행
```bash
# GUI 클라이언트 (권장)
java -cp "bin;lib/sqlite-jdbc-*.jar" LoginFrame

# 콘솔 클라이언트 (테스트용)
java -cp "bin;lib/sqlite-jdbc-*.jar" Client
```
